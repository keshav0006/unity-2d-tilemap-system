#  Unity 2D Tilemap System

A clean implementation of Unity’s 2D Tilemap workflow using Grid, Tilemap,
and Collider components. Focused on level layout and player interaction.


##  Features
- Grid + Tilemap setup
- Tilemap Collider 2D
- Player movement on tiles
- Modular scripts

##  Tools Used
- Unity (2D)
- C#
- Tilemap & Grid
- Rigidbody2D / Collider2D


## Structure
Assets/

 │    ├── GridMovement.cs
